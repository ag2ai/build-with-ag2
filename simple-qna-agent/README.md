# Simple Q&A Agent

一个简单的问答助手，使用 AG2 框架构建。

## 功能

- 用户提问，AI 用简单语言回答
- 适合初学者学习 AG2 的基本用法

## AG2 Features

- AssistantAgent: AI 助手
- UserProxyAgent: 用户代理

## TAGS

TAGS: qa, beginner, agent, deepseek

## 安装

```bash
pip install "ag2[openai]" python-dotenv
```

## 配置

```bash
cp .env.example .env
# 编辑 .env 填入你的 DeepSeek API Key
```

## 运行

```bash
python main.py
```
